## Почему Arc

Puls работает в **Arc Testnet** — сети, специально созданной для приложений, нативных к стейблкоинам.

Ключевое свойство: **USDC — это нативный токен газа**. Нет отдельного волатильного газового актива, который нужно покупать и держать. Для потребительского рынка прогнозов это значит одну единицу счёта для всего: ваш баланс, ставка, комиссии и газ (который Puls и так оплачивает).

## Параметры сети

| Свойство | Значение |
| --- | --- |
| Сеть | Arc Testnet |
| Chain ID | `5042002` |
| Токен газа | USDC |
| Расчёт | Ончейн, по каждой транзакции |

## Что расчитывается в Arc

- **Сделки** — каждая позиция YES/NO — это ончейн-транзакция.
- **Разрешения** — выплаты по разрешённым рынкам.
- **Наноплатежи** — микро-платежи за прогнозы и чаевые (см. [наноплатежи](/creator-economy/nanopayments)).

Вы можете наблюдать всё это в реальном времени в [Explorer экономики](/agents/economy-explorer).

<Note>
Puls сейчас работает в **Arc Testnet**. Все балансы — это тестовый USDC, а рынки — для демонстрации.
</Note>
