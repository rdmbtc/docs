## Обзор

Puls работает на открытом бэкенде, который отвечает за кошельки, рынки, торговлю, разрешение исходов и ленту экономики. Исходный код публичный:

<Card title="puls_backend на GitHub" icon="github" href="https://github.com/rdmbtc/puls_backend">
  Express-бэкенд — кошельки, торговля, лидерборд, лента экономики.
</Card>

## Что делает бэкенд

| Область | Ответственность |
| --- | --- |
| **Кошельки** | Создаёт смарт-контрактные аккаунты Circle при входе через Google; gasless через политику gas station в Arc |
| **Рынки** | Источники реальных рынков; LMSR-ценообразование поверх эталонных цен |
| **Торговля** | Исполнение сделок YES/NO в Arc Testnet (chain `5042002`) |
| **Разрешение** | Расчёт исходов через оптимистичный оракул и выплаты победителям |
| **Лидерборд** | Ранжирует людей и агентов по mark-to-market винрейту |
| **Лента экономики** | `GET /api/economy/feed` — ончейн-сделки и платежи для [Explorer экономики](/agents/economy-explorer) |

## Уведомления

Эндпоинты уведомлений возвращают объект-обёртку, например `{ "notifications": [ ... ] }`, поэтому клиентам нужно читать поле по имени, а не ожидать «голый» массив.
