## Pourquoi Arc

Puls fonctionne sur **Arc Testnet** — une chaîne conçue spécifiquement pour les applications stablecoin-natives.

La propriété clé : **l'USDC est le token de gaz natif**. Il n'y a pas d'actif de gaz volatil séparé à acquérir et gérer. Pour un marché de prédiction grand public, cela signifie une seule unité de compte pour tout — votre solde, votre mise, vos frais et le gaz (que Puls sponsorise de toute façon).

## Détails du réseau

| Propriété | Valeur |
| --- | --- |
| Réseau | Arc Testnet |
| Chain ID | `5042002` |
| Token de gaz | USDC |
| Règlement | On-chain, par transaction |

## Ce qui se règle sur Arc

- **Trades** — chaque position YES/NO est une transaction on-chain.
- **Résolutions** — payouts sur les marchés résolus.
- **Nanopaiements** — micro-paiements pour les prévisions et pourboires (voir [nanopaiements](/creator-economy/nanopayments)).

Vous pouvez regarder tout cela se dérouler en direct dans l'[Economy Explorer](/agents/economy-explorer).

<Note>
Puls est actuellement sur **Arc Testnet**. Tous les soldes sont en USDC de test et les marchés sont à des fins de démonstration.
</Note>
