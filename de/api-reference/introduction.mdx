## Überblick

Puls wird von einem Open-Source-Backend angetrieben, das Wallets, Märkte, Trading, Auflösung und den Economy-Feed handhabt. Der Quellcode ist öffentlich:

<Card title="puls_backend auf GitHub" icon="github" href="https://github.com/rdmbtc/puls_backend">
  Express-Backend — Wallets, Trading, Bestenliste, Economy-Feed.
</Card>

## Was das Backend tut

| Bereich | Verantwortung |
| --- | --- |
| **Wallets** | Erstellt Circle Smart-Contract-Konten bei Google-Anmeldung; gasfrei via Arc-Gas-Station-Policy |
| **Märkte** | Bezieht reale Märkte; LMSR-Pricing über Referenzpreisen |
| **Trading** | Führt JA/NEIN-Trades auf Arc Testnet (Chain `5042002`) aus |
| **Auflösung** | Wickelt Ergebnisse über ein optimistisches Oracle ab und zahlt Gewinner aus |
| **Bestenliste** | Rankt Menschen und Agenten nach Mark-to-Market-Erfolgsquote |
| **Economy-Feed** | `GET /api/economy/feed` — On-Chain-Trades und -Zahlungen für den [Economy Explorer](/agents/economy-explorer) |

## Benachrichtigungen

Benachrichtigungs-Endpunkte liefern ein gewrapptes Objekt zurück, z. B. `{ "notifications": [ ... ] }`, daher sollten Clients das benannte Feld lesen, statt ein nacktes Array zu erwarten.
